# For some reason the curl request has to be in one continious line.
# Furthermore, install curl with choco and then when opening a new terminal enter `Remove-item alias:curl`. The reason being
  Invoke-WebRequest keeps insisting on using its alias curl instead of the newly installed curl. This will be reset everytime you open a new terminal.
# Because double quotes are being used by the command line and mandatory in json on the first name, you will
 have to paste `\` in front of every double quotes within the -d json configuration.
